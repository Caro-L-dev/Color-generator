# Vue3-color-generator


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Install sass
```
npm install -D sass-loader@^10 sass
```

### Install dependencies
```
npm install clipboard --save
```
